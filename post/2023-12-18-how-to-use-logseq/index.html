How to Use Logseq

 发表于 December 18, 2023  |  1 分钟  |  52 个字  |  svtter

logseq 是一种新型的知识工具。开源，免费（暂时），可以自己管理数据。

概念

logseq 底层数据结构使用了 markdown。因此，每次创建新的 tag 或者[[]]的时候，都可以认为创建了一个新的 page。在这里基础上，来进一步理解 logseq。

Definition

  • 关于
  • #tags: 标签，啥时候都能用；使用后要追加解释。

Problems

问题清单。

目录是什么？

#Contents目录是全局的，用于查看所有文档的工具

如何使用插件？

设置好 logseq 的代理服务器，即可使用。

我推荐的插件有:

  • git 用来管理同步的笔记
  • goto today 用于快速定位到今天
  • todo 用于管理待办事项
  • calendar 把 logseq 转换成日历工具
  • toc 用于生成目录

Usage

  • 使用 logseq 整理论文
  • 右上角三个点，可以增强 logseq
    • Use plugins：插件可以增强 logseq 的功能
    • Use theme：主题可以让 logseq 的可读性变得更好
