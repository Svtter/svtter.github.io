手把手教你搭建 vue-cli﻿

2018-12-28
前端
Comments

vuejs 这东西，实不相瞒，学习了很长时间，从看文档到使用，大概有一年的时间。当然，其实上手没有那么难，<script src="vue.js"></script>这样就搞定了。只是想要用的比较优雅，存在一定的难度。

但是硬核玩家肯定不能止步于此。于是就开始了vue-cli的探究。没想到这个探究如此之长，以至于我在很长一段时间内，都没有正儿八经的写过前端。

安装 nodejs

首先第一步，在ubuntu下安装nodejs。Ubuntu自带的vuejs版本说不过去，版本有些老旧，而且安装相关的包很不方便，需要sudo什么的。因此，建议直接从官网下载。

从官网下载，直接解压的到~/Application/nodejs下面即可。（笔者习惯将已经安装好的软件放在$HOME/Application这个文件夹下面。）然后，将nodejs的执行路径加入到 shell 的运行路径中。

export PATH=$HOME/Application/nodejs:$PATH

如此一来，就把nodejs安装上了。

安装 vue-cli

国内的话最好是先走一下 taobao 的 npm 镜像（同理，其他的技术软件都可以走镜像，土豪公司vpn请无视）。

安装好 vue-cli 以后，就可以开始你的编程之路了。
