Deployment of Dify 1.2.0.md

dify 是一个 ai agent 工具。

Apr 22, 2025
阅读时长: 1 分钟

我认为 hacker 应该抛弃从代码构建 agent 想法，全面拥抱工作流平台，例如 dify。这样的效率比写代码高许多倍。如果一定要去写代码，可以写插件嵌入到 dify。

部署方法

在服务器上执行下面的代码即可。

2
3
4

git clone https://github.com/langenius/dify
cd dify/docker
cp .env.example .env
docker compose up -d
