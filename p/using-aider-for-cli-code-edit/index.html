Using Aider for Cli Code Edit

Jun 03, 2025
阅读时长: 4 分钟

这篇文章是使用 aider 进行命令行文章编辑。

某种程度上，aider 是 openhands 的替代品，成熟度比 openhands 要高很多（开源版本的 openhands 经常不可用）。

相似的产品还有

  • openai codex
  • claude code

安装

安装 aider 非常简单，就像是官方文档描述的一样：

1
2

python -m pip install aider-install
aider-install

配置

然后最好在自己的终端里面配置好 api key。如果使用硅基流动的 deepseek，你可以这样配置：

1
2
3

# using siliconflow
export OPENAI_API_KEY=<your key>
export OPENAI_BASE_URL=https://api.siliconflow.cn/v1
  • 可以使用硅基流动的免费增额。使用我的注册码来注册。
  • 如果需要更高的访问频次，可以直接使用 deepseek 官方的 API。这里可以获取 API KEY。

这样一来，就可以直接使用 aider 了。

下面，我们以硅基流动的 API KEY 为例子，让我们试试采用deepseek-ai/DeepSeek-R1-0528-Qwen3-8B模型。

aider --model openai/deepseek-ai/DeepSeek-R1-0528-Qwen3-8B README.md

本文的 tag 就是通过 aider 生成的。在示例中，我编辑了README.md文件。但是你完全可以打开一个 git repository，来编辑其他的代码。

补充

后来我又使用claude相关的模型来测试了一下 aider 的效果。

aider --model openrouter/anthropic/claude-sonnet-4 --yes

我在 libpaper 项目中开了一个新的分支 feat/web-api，完全由 claude+aider 进行开发。

效果非常好。
